# 🤖 CrewAI & Gemini - Live Gmail Automation Agent

An AI-powered multi-agent system built using **CrewAI** and **Google Gemini 2.5 Flash** that connects to a live Gmail inbox, classifies incoming unread emails (e.g., HR requests, inquiries, complaints), and automatically drafts context-aware responses directly into the Gmail Drafts folder.

---

## 🚀 Features
- **Live IMAP Integration:** Fetches real-time unread emails from Gmail safely.
- **Multi-Agent Architecture:** - *Email Classifier Agent:* Analyzes intent, mood, and urgency level.
  - *Corporate Copywriter Agent:* Drafts natural, human-like, and professional responses.
- **Automated Draft Creation:** Pushes responses directly to your Gmail Drafts folder for final review.
- **Zero Cost Production:** Completely powered by Gemini's free tier API.
